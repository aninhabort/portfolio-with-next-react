# add-project Skill — Debug/Error Fix

## Purpose
Checklist e orientações para depuração e correção de erros ao adicionar projetos ao arquivo de portfólio.

## Itens
- Verifique se todos os campos obrigatórios foram preenchidos.
- Confirme se a categoria está em PROJECT_FILTERS.
- Certifique-se de que não há duplicidade de títulos.
- Valide URLs se fornecidas.
- Garanta que o objeto segue a interface ProjectArchiveEntry.
- Se houver erro de sintaxe, revise a estrutura do objeto inserido.

## Dica
Se o erro persistir, compare com exemplos válidos já presentes no arquivo archive.ts.
