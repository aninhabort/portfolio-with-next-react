# add-project Skill — Feature Builder

## Objetivo
Checklist para garantir que a adição de um novo projeto ao portfólio siga o fluxo correto e cubra todos os requisitos de negócio e UI.

## Itens
- Solicitar todos os dados obrigatórios do projeto.
- Validar categoria e tags.
- Gerar objeto conforme interface ProjectArchiveEntry.
- Inserir no topo do array PROJECT_ARCHIVE.
- Manter formatação e ordem alfabética das chaves.
- Confirmar inserção e mostrar snippet atualizado ao usuário.

## Dica
Sempre siga o padrão de formatação do arquivo para evitar erros de lint ou build.
