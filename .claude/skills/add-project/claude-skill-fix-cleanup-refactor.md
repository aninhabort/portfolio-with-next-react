# add-project Skill — Fix/Cleanup/Refactor

## Objetivo
Checklist para ajustes, refatoração e limpeza ao adicionar ou atualizar projetos no arquivo de portfólio.

## Itens
- Remover entradas duplicadas.
- Corrigir campos inconsistentes (ex: nomes de chaves, tipos de dados).
- Padronizar ordem das chaves dos objetos.
- Garantir uso de aspas duplas e vírgulas finais.
- Atualizar ou remover projetos obsoletos, se necessário.

## Dica
Após alterações, rode o lint e build do projeto para garantir que não há erros.
