# add-project Skill — UI Consistency

## Objetivo
Checklist para garantir consistência visual e de experiência ao adicionar novos projetos ao portfólio.

## Itens
- Verifique se a descrição está clara e objetiva.
- Certifique-se de que as tags são relevantes e padronizadas.
- Use imagens de preview com proporção e resolução adequadas.
- Mantenha o padrão de cores e tipografia do portfólio.
- Revise o texto para evitar repetições e erros gramaticais.

## Dica
Compare o novo projeto com os já existentes para manter a harmonia visual e textual.
